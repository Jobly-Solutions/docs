Los usuarios pueden subir archivos durante una conversación con un agente. Esta función es útil cuando los usuarios necesitan compartir archivos con agentes o con operadores humanos.

<Note>
  Los archivos subidos durante una conversación no son accesibles para otras
  conversaciones. Para extender el conocimiento del Scout para todas las conversaciones, usa un
  [Almacén de Datos](/datastore) en su lugar.
</Note>

{/* ![ejemplo 1](/images/conversation-file-upload/1.jpg) */}

Los agentes pueden procesar los siguientes tipos de archivo:

- text/csv
- text/plain
- text/markdown
- application/pdf
- application/json
- application/vnd.openxmlformats-officedocument.presentationml.presentation
- application/vnd.openxmlformats-officedocument.wordprocessingml.document
- application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
- image/png (Se requiere modelo compatible con visión como GPT-4-Turbo o Claude 3)
- image/jpeg (Se requiere modelo compatible con visión como GPT-4-Turbo o Claude 3)
- image/gif (Se requiere modelo compatible con visión como GPT-4-Turbo o Claude 3)
- image/webp (Se requiere modelo compatible con visión como GPT-4-Turbo o Claude 3)

<Warning>Los tipos de archivo no soportados serán ignorados por la IA</Warning>

Para reutilizar un archivo subido usa el menú desplegable de archivos de conversación
