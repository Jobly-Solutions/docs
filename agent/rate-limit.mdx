Puedes configurar un límite de velocidad desde la pestaña de configuración global de tu Scout para limitar el número de mensajes que tus usuarios pueden enviar.

**Esta configuración se aplica solo a las siguientes integraciones**

- Widget de Burbuja de Chat
- Widget de iFrame
- Widget de Página Independiente
