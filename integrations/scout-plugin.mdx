## Preguntas Frecuentes

### ¿Cómo instalar un plugin privado?

Instalar un plugin privado es sencillo. Solo sigue el video de arriba para instalar un Plugin no oficial

### ¿Cómo publicar el plugin para hacerlo disponible para todos los usuarios de Scout?

Para publicar un plugin, necesitas enviar una solicitud a través del bot de envío de plugins.
**Ten en cuenta que tu almacén de datos necesita ser público para hacer el plugin disponible para todos los usuarios de Scout.**

[Aprende más](https://platform.openai.com/docs/plugins/review/submit-a-plugin-for-review)

### El plugin no es detectado por Scout

Si el plugin no es detectado por Scout, podrías necesitar optimizar el campo "descripción para humanos" de tu plugin.

[Mejores prácticas para descripción de plugins](https://platform.openai.com/docs/plugins/getting-started/writing-descriptions)
