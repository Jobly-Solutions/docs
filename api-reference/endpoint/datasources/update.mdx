### Ruta

<ParamField path="datastoreId" type="string" required>

</ParamField>

### Cuerpo

<ParamField body="id" type="string" required>
  Este es el ID de la Fuente de Datos que quieres actualizar.
</ParamField>

<ParamField body="text" type="string" required>
  Este es el nombre de la Fuente de Datos.
</ParamField>

<ParamField body="name" type="string">
  Este es el nombre de la Fuente de Datos.
</ParamField>

<ParamField body="metadata" type="JSON o null">
  Este es el objeto de metadatos de la Fuente de Datos.
</ParamField>
