---
title: 'Fuente de Datos - Upsert (⚠️ obsoleta)'
api: 'POST https://app.braviloai.com/api/datastores/upsert/{datastoreId}'
description: 'Este endpoint te permite agregar una nueva Fuente de Datos en un Almacén de Datos'
authMethod: 'bearer'
---

### Ruta

<ParamField path="datastoreId" type="string" required>
  El id del almacén de datos al que quieres acceder.
</ParamField>

### Cuerpo

<ParamField path="documents" type="array" required>
  
<Expandable title="Alternar objeto" defaultOpen>

Los documentos que quieres hacer upsert.

<ParamField body="text" type="string" required>
  Este es el nombre de la Fuente de Datos.
</ParamField>

<ParamField body="name" type="string">
  Este es el nombre de la Fuente de Datos.
</ParamField>

<ParamField body="metadata" type="JSON o null">
  Este es el objeto de metadatos de la Fuente de Datos.
</ParamField>

</Expandable>

</ParamField>

### Respuesta

<ResponseField name="ids" type="array de strings">
  Los IDs de los documentos que fueron creados.
</ResponseField>

<RequestExample>

```bash Ejemplo de Solicitud
curl --location --request POST 'https://app.braviloai.com/api/datastores/upsert/<datastoreId>' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API_KEY>' \
--data-raw '{
    "documents": [
      "text": "Lorem Ipsum...",
      "name": "Mi Documento",
      "metadata": {
        "source" : "https://en.wikipedia.org/wiki/Lorem_ipsum",
        "author": "John Doe",
        "date": "2021-01-01",
      }
    ]
}'
```

</RequestExample>

<ResponseExample>

```json Respuesta
{
  "ids": ["60f9b9f0-0b1a-11ec-9a03-0242ac130003"]
}
```

</ResponseExample>
