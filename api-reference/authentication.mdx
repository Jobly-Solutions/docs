---
title: 'Autenticación'
description: 'Esto demuestra cómo autenticar una solicitud a la API de braviloai.'
---

Para acceder a un Scout o Almacén de Datos privado, debes proporcionar una clave API de braviloai en el encabezado de Autorización de tu solicitud.

```bash
'Authorization': 'Bearer <API_KEY>'
```

## Claves API

#### Puedes obtener tu clave API desde la página de configuración de tu cuenta.

https://app.braviloai.com/account
