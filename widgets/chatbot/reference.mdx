## Atributos / Props

<Tabs>
  <Tab title="HTML" >
    ```html
    <script type="module">
  import Chatbox from 'https://cdn.jsdelivr.net/npm/@bravilo/embeds/dist/chatbox/index.js';

Chatbox.initBubble({
agentId: 'TU_SCOUT_ID',

    // Si se proporciona creará un contacto para el usuario y lo vinculará a la conversación
    contact: {
    firstName: 'John',
    lastName: 'Doe',
    email: 'customer@email.com',
    phoneNumber: '+33612345644',
    userId: '42424242',
    },
    // Sobrescribir mensajes iniciales
    initialMessages: [
    'Hola Georges, ¿cómo estás hoy?',
    '¿Cómo puedo ayudarte?',
    ],
    // El contexto proporcionado se agregará al prompt del sistema del Scout
    context: "El usuario con quien estás hablando es John. Comienza saludándolo por su nombre.",

});

</script>
    ```

## Métodos

<ParamField path="open" type="solo widget de burbuja">
Abrir Chatbox de Burbuja
```js
const widget = await Chatbox.initBubble({
  agentId: 'TU_SCOUT_ID',
})

widget.open()

`````
</ParamField>

<ParamField path="close"   type="solo widget de burbuja">
  Cerrar Chatbox de Burbuja
  ```js
  const widget = await Chatbox.initBubble({
    agentId: 'TU_SCOUT_ID',
  })

  widget.close()
```

</ParamField>

<ParamField path="toggle"  type="solo widget de burbuja">

```js Alternar Chatbox de Burbuja
const widget = await Chatbox.initBubble({
  agentId: 'TU_SCOUT_ID',
})

widget.toggle()
```
</ParamField>


<ParamField path="createNewConversation"   >
```js Crear Nueva Conversación
const widget = await Chatbox.initBubble({
  agentId: 'TU_SCOUT_ID',
})

widget.createNewConversation()
```
