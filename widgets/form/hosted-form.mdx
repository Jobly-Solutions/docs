---
description: 'Formulario Alojado de Bravilo AI'
---

Todos los formularios están disponibles desde la URL Alojada de Bravilo AI.

```bash
https://app.braviloai.com/forms/TU_FORMULARIO_ID
```

<Note>
  Puedes encontrar el ID del Formulario en la configuración del Formulario bajo la pestaña de Configuración.
</Note>

La forma más fácil de comenzar es simplemente redirigir al usuario a esta página desde dentro de tu aplicación o sitio web.
